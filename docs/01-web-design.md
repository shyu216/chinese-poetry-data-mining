# 墨韵诗海 · Web 创意设计方案

> **创意愿景**：数字园林中的诗词漫步  
> **核心氛围**：让 33 万首诗词在数字世界中重新呼吸  
> **最后更新**: 2026-03-15

---

## 🎭 氛围调性系统

### 整体感觉
不是冰冷的数据库，而是**有温度、有呼吸的文化空间**。用户不是"查询者"，而是"踏入藏书阁的旅人"。

### 色彩系统 · 水墨意境

| 色彩角色 | 色值 | 意象 | 使用场景 |
|---------|------|------|---------|
| **主色-赭石** | `#8B4513` | 古卷、宣纸、印章 | 标题、重点、按钮 |
| **辅色-墨青** | `#2F4F4F` | 水墨、远山、深邃 | 正文、次级信息 |
| **点缀-朱砂** | `#C41E3A` | 印章、批注、重要 | 高亮、选中、徽章 |
| **背景-宣纸** | `#F5F5DC` | 宣纸质感、温暖 | 页面背景 |
| **淡墨** | `#D4C5B5` | 淡墨渲染、层次 | 边框、分割线 |
| **烟灰** | `#8B8682` | 古旧、沉淀 | 禁用、提示文字 |

### 朝代专属色

| 朝代 | 色彩 | 意象 |
|-----|------|------|
| 唐 | `#C41E3A` 朱砂红 | 盛世气象、热烈奔放 |
| 宋 | `#1E40AF` 青花蓝 | 理性雅致、内敛深沉 |
| 元 | `#047857` 松石绿 | 草原气息、清新自然 |
| 明 | `#B45309` 琥珀棕 | 稳重典雅、大气磅礴 |
| 清 | `#7C3AED` 紫罗兰 | 精致繁复、哀婉深沉 |

### 字体系统

| 用途 | 字体 | 备选 |
|-----|------|------|
| **标题** | 思源宋体 / 方正清刻本悦宋 | 衬线体，有书法韵味 |
| **正文** | 思源宋体 | 清晰易读 |
| **数据** | JetBrains Mono | 等宽，但温暖 |
| **诗词内容** | 方正苏新诗柳楷 | 楷体，古典感 |

### 动效原则

| 原则 | 描述 | 实现 |
|-----|------|------|
| **缓入缓出** | 如毛笔起落，不突兀 | `cubic-bezier(0.4, 0, 0.2, 1)` |
| **留白呼吸** | 元素间有"气口"，不拥挤 | 适当的 margin/padding |
| **微交互** | 悬停如指尖触纸，轻微反馈 | transform + opacity |
| **墨迹流动** | 内容出现如墨晕染开 | clip-path / mask 动画 |

## 🏛️ 五大 View 设计

### 1. 首页 · 诗意入口 (HomeView)

**氛围**：晨雾中的古卷缓缓展开

#### 交互亮点
- **时间感知问候**：根据时段显示"晨安/午安/晚安，寻诗者"
- **墨迹搜索框**：输入时笔触跟随，如毛笔书写
- **古卷翻页**：随机诗词展示有翻页动效
- **飘落效果**：搜索结果如花瓣/落叶飘落呈现

#### 晨昏变化
| 时段 | 背景色调 | 问候语 |
|-----|---------|--------|
| 5-8点 | 淡金晨曦 | "晨安，寻诗者" |
| 8-12点 | 明亮日光 | "上午好，寻诗者" |
| 12-14点 | 温暖午后 | "午安，寻诗者" |
| 14-18点 | 柔和夕照 | "下午好，寻诗者" |
| 18-22点 | 暮色苍茫 | "晚安，寻诗者" |
| 22-5点 | 深夜静谧 | "夜深了，寻诗者" |

---

### 2. 词相似度 · 语义星云 (WordSimilarityView)

**氛围**：词语在宇宙中形成星座，相似词如行星环绕

#### 交互亮点
- **力导向网络**：中心词如恒星，相似词如行星环绕
- **无缝探索**：点击任意词成为新的中心，平滑过渡动画
- **引力效果**：节点有物理模拟，可拖拽、有弹性
- **诗词联动**：选中词后，下方浮现含该词的名句

#### 视觉层次
| 相似度 | 视觉表现 | 颜色 |
|-------|---------|------|
| 0.90+ | 大节点、粗连线 | `#2ecc71` 翠绿 |
| 0.80-0.89 | 中节点、中连线 | `#3498db` 天蓝 |
| 0.75-0.79 | 小节点、细连线 | `#f39c12` 琥珀 |
| < 0.75 | 微节点、虚线 | `#e74c3c` 淡红 |

---

### 3. 作者页 · 诗人画像 (AuthorsView)

**氛围**：历代文人的数字纪念馆，诗坛座次可视化

#### 交互亮点
- **诗坛座次**：排行榜如古代科举榜单，有卷轴展开动效
- **诗人卡片**：头像 + 代表作一句 + 创作数量徽章
- **相似作者**："与李白神交的诗人们" —— 基于词频相似度推荐
- **时间轴**：作者活跃时期在朝代轴上的位置

#### 排名徽章
| 排名 | 徽章 | 颜色 |
|-----|------|------|
| 1 | 🥇 状元 | 金色 `#FFD700` |
| 2 | 🥈 榜眼 | 银色 `#C0C0C0` |
| 3 | 🥉 探花 | 铜色 `#CD7F32` |
| 4-10 | 🏅 进士 | 青色 `#18a058` |
| 11-50 | 📜 举人 | 灰色 `#8b7355` |

---

### 4. 统计页 · 数据诗学 (StatsView)

**氛围**：用数据讲述诗歌史，可视化如山水画卷

#### 交互亮点
- **时间河流**：朝代如河流，诗词数量如水流密度
- **环形图**：如年轮，一圈圈展开诗歌史
- **词演变动画**：看一个词从唐到宋到清的频率变化
- **点击下钻**：点击图表元素，跳转到对应筛选的诗词列表

---

### 5. 词云页 · 语言地形 (WordcloudView)

**氛围**：俯瞰整个时代的语言地貌，词如山川起伏

#### 交互亮点
- **动态词云**：词如落叶，有轻微飘动效果
- **朝代对比**：唐宋词云并置，一眼看出"气象"差异
- **点击探索**：点击词进入"词境"，展示该词在不同时代的演变
- **意境标签**：不仅显示频率，还显示该词在不同时代的"意境标签"

## 🛠️ 技术实现建议

### 动效实现

| 效果 | 技术方案 |
|-----|---------|
| 墨迹晕染 | CSS `mask-image` + 动画 |
| 力导向网络 | D3.js `forceSimulation` |
| 词云布局 | D3-cloud |
| 翻页效果 | CSS 3D transform |
| 飘落效果 | Canvas 或 CSS animation |

### 性能优化

| 策略 | 实现 |
|-----|------|
| 虚拟滚动 | 长列表只渲染可视区域 |
| 分片加载 | 按需加载诗词数据 |
| 索引缓存 | LocalStorage/IndexedDB |
| Web Worker | 大数据计算不阻塞主线程 |
