## 页面概述

诗词浏览页面，采用**渐进式加载**策略处理 332,712 首诗词的大数据量。

---

## 核心设计决策

### 1. 分块加载架构

**问题**: 33万首诗词无法一次性加载  
**方案**: 将数据分成 333 个 chunks，每 chunk 约 1000 首

```
初始加载: 1 chunk (~1000首)
用户翻页: 从已加载 chunks 中筛选分页
到达末页: 自动加载下一 chunk
手动加载: 点击"加载更多"按钮
```

### 2. 顶部统计栏

```
332,712 总收录 | 1/333 已加载分块 | 0.3% 数据量
```

- **总收录**: 数据库中诗词总数
- **已加载分块**: 当前已加载的 chunk 数量 / 总数
- **数据量**: 已加载数据占比（绿色提示还有数据）

### 3. 筛选区域

```
[朝代下拉] [体裁下拉] [搜索框] [筛选按钮] [清除按钮]
```

筛选逻辑：
- 从**已加载**的 chunks 中过滤
- 筛选结果过少时，需要手动加载更多 chunks

### 4. 诗词卡片（紧凑设计）

**布局**: 单行横向排列

```
[唐]  静夜思          >
      李白 · 诗
```

**元素**:
- 朝代标签：彩色背景，显示朝代名
- 标题：单行截断，15px 加粗
- 副标题：作者 · 体裁，13px 灰色
- 箭头：悬停时淡入并向右滑动

**样式**:
```css
.poem-card-compact {
  padding: 12px 16px;
  border: 1px solid var(--color-border);
  border-radius: 8px;
}
```

### 5. 加载更多区域

**三种状态**:

1. **平时状态**:
   ```
   [加载更多诗词 (已加载 1,000 首)]
   每批加载约 1000 首诗词
   ```

2. **最后一页提示**:
   ```
   [加载更多诗词 (已加载 1,000 首)]
   💡 提示：翻到最后一页会自动加载下一批
   ```

3. **自动加载中**:
   ```
   [🔄 已翻到最后一页，正在自动加载下一批...]
   ```

### 6. 分页器

- 按**已加载**的诗词数量计算页数
- 初始约 10-40 页（取决于 pageSize）
- 支持 pageSize 切换: 12, 24, 48, 96

---

## 交互流程

```
用户进入页面
    ↓
自动加载第 1 个 chunk (~1000首)
    ↓
显示第 1 页诗词（如 24首/页）
    ↓
用户翻页 → 从已加载数据分页
    ↓
到达最后一页
    ↓
自动加载第 2 个 chunk
    ↓
页数增加，继续浏览
```

## 技术实现

### 状态管理

```typescript
// 核心状态
const loadedChunkIds = ref<number[]>([])  // 已加载的 chunk IDs
const totalChunksCount = ref(333)         // 总 chunk 数
const page = ref(1)                       // 当前页
const pageSize = ref(24)                  // 每页数量

// 计算属性
const loadedPoemsCount = computed(() => loadedChunkIds.value.length * 1000)
const hasMoreChunks = computed(() => loadedChunkIds.value.length < totalChunksCount.value)
const isLastPage = computed(() => page.value >= Math.ceil(totalCount.value / pageSize.value))
```

### 加载逻辑

```typescript
// 初始加载
const initialLoad = async () => {
  const relevantChunks = await getRelevantChunkIds()
  if (relevantChunks.length > 0) {
    await loadChunkSummaries(relevantChunks[0])
    loadedChunkIds.value = [relevantChunks[0]]
  }
  await loadPoems()
}

// 加载下一 chunk
const loadNextChunks = async () => {
  const relevantChunks = await getRelevantChunkIds()
  const nextChunkId = relevantChunks.find(id => !loadedChunkIds.value.includes(id))
  if (nextChunkId !== undefined) {
    await loadChunkSummaries(nextChunkId)
    loadedChunkIds.value.push(nextChunkId)
    await loadPoems()
  }
}

// 自动加载（到达最后一页时）
const handlePageChange = async (p: number) => {
  page.value = p
  await loadPoems()
  if (isLastPage.value && hasMoreChunks.value) {
    await loadNextChunks()
  }
}
```

### 筛选逻辑

```typescript
const loadPoems = async () => {
  // 从已加载 chunks 获取所有诗词
  const allLoadedPoems: PoemSummary[] = []
  for (const chunkId of loadedChunkIds.value) {
    const chunkPoems = await loadChunkSummaries(chunkId)
    allLoadedPoems.push(...chunkPoems)
  }
  
  // 应用筛选
  let filtered = allLoadedPoems
  if (dynastyFilter.value) {
    filtered = filtered.filter(p => p.dynasty === dynastyFilter.value)
  }
  if (genreFilter.value) {
    filtered = filtered.filter(p => p.genre === genreFilter.value)
  }
  if (searchQuery.value.trim()) {
    const query = searchQuery.value.toLowerCase()
    filtered = filtered.filter(p => 
      p.title?.toLowerCase().includes(query) ||
      p.author?.toLowerCase().includes(query)
    )
  }
  
  // 分页
  const start = (page.value - 1) * pageSize.value
  poems.value = filtered.slice(start, start + pageSize.value)
  totalCount.value = filtered.length
}
```

## 性能优化

1. **按需加载**: 只加载用户需要的 chunks
2. **缓存机制**: 已加载的 chunks 缓存到内存
3. **本地筛选**: 筛选在客户端完成，无需重复请求
4. **分页渲染**: 每页最多 96 首，避免 DOM 过载
