# 2026-03-15 Poems页面设计

## 核心挑战

332,712首诗词无法一次性加载。

## 解决方案：渐进式分块加载

```
初始加载: 1 chunk (~1000首)
用户翻页: 从已加载chunks中筛选
到达末页: 自动加载下一chunk
```

## 页面结构

**顶部统计栏**:
```
332,712 总收录 | 1/333 已加载分块 | 0.3% 数据量
```

**诗词卡片**（紧凑设计）:
```
[唐]  静夜思          >
      李白 · 诗
```

**加载状态**:
- 平时：显示"加载更多"
- 末页：提示自动加载
- 加载中：显示loading动画

## 技术要点

- 分页器按已加载数量计算
- 筛选从已加载数据中过滤
- 支持pageSize切换: 12/24/48/96
