# 2026-03-14 项目架构总览

## 核心设计理念

**完全去后端化**：所有分析在构建时完成，客户端执行搜索。

## 五层数据架构

```
RAW → BRONZE → SILVER → GOLD → OUTPUT
```

| 层级 | 说明 | 输出 |
|------|------|------|
| Bronze | 清洗合并 | v1_poems_merged.csv |
| Silver | 结构化、格律提取 | v2_poems_*.csv |
| Gold | 相似度、词频分析 | v3_*.json |
| Output | Web就绪静态文件 | JSON分片 + HTML |

## 核心管线

1. **01_clean.py** - 数据清洗、繁简转换、去重
2. **02_structure.py** - 格律提取、诗体分类
3. **03_analyze_words.py** - jieba分词、词频统计
4. **04_analyze_similarity.py** - TF-IDF、余弦相似度

## 关键模块

- **ChunkManager** - 分块处理大规模数据
- **WordFrequencyAnalyzer** - 词汇频率分析
- **TextSimilarityAnalyzer** - 文本相似度计算

## 技术亮点

- 纯静态部署（GitHub Pages）
- 构建时计算
- 客户端倒排索引搜索
- 分块处理降低内存占用
- Bronze/Silver/Gold分层管理
