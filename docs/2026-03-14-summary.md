## 🎨 前端技术栈

### 核心技术

- **原生 JavaScript** - 不使用 React/Vue 等框架

- **渐进式加载** - 分片加载诗词数据
- **虚拟滚动** - 最多渲染 100 个 DOM 节点

## 🏗️ 架构特点

### 静态部署架构

```
构建时计算 → 静态 JSON 索引 → 客户端搜索 → GitHub Pages 部署
```

### 数据流

1. **构建阶段**: Python 脚本分析数据，生成静态索引
2. **部署阶段**: 上传静态文件到 GitHub Pages
3. **运行阶段**: 浏览器加载索引，执行搜索和可视化

### 性能优化

- **分片加载**: 诗词数据分成多个 JSON 文件
- **虚拟滚动**: 只渲染可见区域的 DOM 节点
- **搜索缓存**: 使用 Map 缓存搜索结果
- **懒加载**: 按需加载诗词详情
- **节流处理**: 滚动事件节流

## 🎯 核心设计理念

这个项目最大的特点是**完全去后端化**：

- ❌ 无需 Flask/Django 服务器
- ❌ 无需 MySQL/PostgreSQL 数据库
- ❌ 无需实时计算
- ✅ 所有分析在构建时完成
- ✅ 客户端执行所有搜索
- ✅ 预计算的静态索引

这种架构的优势是**零成本部署**（GitHub Pages 免费托管）、**极快的响应速度**（无需网络请求）、**无限扩展性**（CDN 缓存）。

## 🔄 数据处理步骤

### 步骤 1: 数据清洗 (01\_clean.py)

功能：

- 加载 chinese-poetry submodule 的原始数据（全唐诗、宋词）
- 统一诗/词/曲字段名
- 繁简转换
- 基于内容 hash 去重
- 生成采样数据
  关键技术：
- JSON 文件批量加载
- MD5 hash 去重
- Pandas DataFrame 处理
  输出： data/bronze/v1\_poems\_merged.csv

### 步骤 2: 结构化处理 (02\_structure.py)

功能：

- 解析诗句结构（按行分割）
- 提取格律模式（如 "7,7,7,7" 表示七言绝句）
- 判断是否为格律诗（绝句、律诗）
- 分类诗体类型（五言绝句、七言律诗等）
- 使用分块处理 （ChunkManager）处理大规模数据
  关键技术：
- 正则表达式清理文本
- 格律模式识别
- 诗体分类算法
- ChunkManager 分块管理（每块 1000 首诗词）
  输出： data/silver/chunks/v2\_poems\_\*.csv （分片存储）

### 步骤 3: 词汇频率分析 (03\_analyze\_words.py)

功能：

- 使用 jieba 分词统计每位作者的高频词汇
- 词性标注（名词、动词、形容词）
- 排除停用词（虚词、助词等）
- 生成全局词频统计
- 支持断点续传 （中间状态缓存）
  关键技术：
- jieba 分词 + 词性标注
- Counter 词频统计
- 停用词过滤
- 中间状态缓存（每 10000 条保存一次）
  输出： data/gold/v3\_word\_frequency.json

### 步骤 4: 文本相似度分析 (04\_analyze\_similarity.py)

功能：

- 为每位作者构建文档（所有诗词拼接）
- 使用 TF-IDF 向量化
- 计算 Cosine Similarity （余弦相似度）
- 构建相似度矩阵
- 提取每位作者的 Top N 相似作者
- 构建网络边（用于 D3.js 可视化）
  关键技术：
- scikit-learn 的 TfidfVectorizer
- Cosine Similarity 计算
- 相似度矩阵构建
- 网络图数据生成
  输出： data/gold/v3\_text\_similarity.json

## 🧩 核心模块

### ChunkManager 分块管理器 (chunk.py)

用于处理大规模诗词数据的分块管理：

- split\_to\_chunks() : 将 DataFrame 分割成多个 chunk 文件（每块 1000 条）
- read\_chunk() : 读取指定 chunk
- read\_all\_chunks() : 读取所有 chunk 并合并
- iter\_chunks() : 迭代所有 chunk（内存友好）
- get\_chunk\_count() : 获取 chunk 总数
  优势：
- 降低内存占用
- 支持增量处理
- 便于并行处理

### WordFrequencyAnalyzer 词汇频率分析器 (word\_frequency.py)

- 使用 jieba 分词 + 词性标注
- 统计每位作者的 Top N 高频词
- 过滤停用词和短词
- 生成词性分布
- 支持作者用词习惯比较

### TextSimilarityAnalyzer 文本相似度分析器 (text\_similarity.py)

- TF-IDF 向量化作者文档
- Cosine Similarity 计算相似度
- 构建相似度矩阵
- 提取相似作者列表
- 生成网络边数据

## 📦 索引构建（静态化）

在数据分析完成后，运行索引构建脚本生成静态索引：

1. build\_search\_index.py - 构建搜索索引（倒排索引）
2. build\_author\_similarity\_index.py - 构建作者相似度索引
3. build\_word\_similarity\_index.py - 构建词汇相似度索引
4. build\_pos\_index.py - 构建词性索引
5. build\_recommendation\_index.py - 构建推荐索引
6. build\_all\_indexes.py - 一键构建所有索引
