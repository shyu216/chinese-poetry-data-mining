## 部署计划

### 部署方式
- **GitHub Pages**: 纯静态部署
- **CI/CD**: GitHub Actions 自动化构建

### 部署步骤

1. **分支合并**
   ```bash
   git checkout main
   git merge refactor/pipeline-v2
   git push origin main
   ```

2. **触发 CI/CD**
   - 自动运行 lint, test, pipeline, deploy 四个 Job
   - 构建完整数据集索引
   - 部署到 GitHub Pages

3. **验证部署**
   - 访问 https://<username>.github.io/chinese-poetry-data-mining
   - 测试搜索功能
   - 验证相似度查询
   - 检查推荐系统

### 回滚计划
```bash
# 如果部署失败，回滚到上一个版本
git revert HEAD --no-edit
git push origin main
```
