## 📊 项目概览

基于 chinese-poetry 开源数据集，构建33万首古典诗词的数据挖掘与可视化平台。

### 核心数据
| 指标 | 数值 |
|------|------|
| 诗词总数 | 332,602 首 |
| 格律模式 | 16,471 种 |
| 高频格律 (≥100首) | 113 种 |
| 覆盖率 | 88.8% |
| 朝代覆盖 | 唐、宋、元、明、清等 |
| 体裁分布 | 诗、词、曲 |

## 🚀 快速开始

### 创建环境

```bash
# 创建并激活 conda 环境
conda env create -f environment.yml
conda activate chinese-poetry-data-mining
```

### 本地开发

```bash
# 1. 运行数据处理
python scripts/01_data_process.py

# 2. 运行分析脚本
python scripts/02_analysis_sentiment.py --data sample
python scripts/02_analysis_network.py --data sample
python scripts/02_analysis_meter.py --data sample

# 3. 生成可视化
python scripts/03_vis_sentiment.py --data sample
python scripts/03_vis_network.py --data sample
python scripts/03_vis_meter.py --data sample

# 4. 构建索引
python scripts/04_build_index.py --data sample

# 5. 本地预览
python scripts/04_serve.py --dir reports/sample/visualizations

# 6. 启动 API 服务
python scripts/05_api_server.py

# 7. 启动静态文件服务器
cd data/github_pages
python -m http.server 8080

# 8. 访问
# http://localhost:8080/index.html
```

### 部署到 GitHub Pages

```bash
# 1. 导出数据
python scripts/08_export_github_pages.py

# 2. 推送到 GitHub
git add .
git commit -m "update: code refactoring and environment configuration"
git push origin main

# 3. GitHub Actions 会自动部署
# 访问: https://yourusername.github.io/chinese-poetry-data-mining
```
