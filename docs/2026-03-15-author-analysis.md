# 诗人数据处理与分析 - 开发方案

## 概述

本方案用于处理诗人相关数据，收集诗人所有作品，分析格律和用词喜好，并设计诗人相似度计算方法。

---

## 输入数据

- **目录**: `results/preprocessed/poems_chunk_*.csv`
- **格式**: CSV 文件，包含以下字段：
  - `id`: 诗词唯一标识
  - `title`: 标题
  - `author`: 作者（繁体）
  - `dynasty`: 朝代
  - `genre`: 体裁（诗/词）
  - `poem_type`: 诗体类型（如七言绝句、五言律诗等）
  - `sentences_simplified`: 简体诗句
  - `meter_pattern`: 格律模式（如 "7,7,7,7"）
  - `hash`: 诗词hash
  - `words`: 分词结果

---

## 输出数据

- **目录**: `results/author/`

### 1. author_summary.json

诗人按作品数量排序列表，每个单元包含：
- `author`: 诗人名字（简体）
- `poem_ids`: 诗词id列表

```json
[
  {
    "author": "李白",
    "poem_ids": ["id1", "id2", ...]
  },
  {
    "author": "杜甫",
    "poem_ids": ["id3", "id4", ...]
  }
]
```

### 2. author_chunk_*.json（高产诗人，每人一个文件）

**高产诗人定义**: 作品数 > 100 首

**文件命名**: 按作品数量降序，序号从 0 开始

**文件格式**:
```json
{
  "author": "李白",
  "poem_type_counts": {
    "七言绝句": 10,
    "五言律诗": 15
  },
  "meter_patterns": [
    {"pattern": "7,7,7,7", "count": 10},
    {"pattern": "5,5,5,5", "count": 15}
  ],
  "poem_ids": ["id1", "id2", ...],
  "word_frequency": {
    "明月": 50,
    "清风": 45,
    ...
  },
  "similar_authors": [
    {"author": "杜甫", "similarity": 0.65},
    {"author": "王维", "similarity": 0.52}
  ]
}
```

### 3. author_chunk_*.json（低产诗人，多人一文件）

**低产诗人定义**: 作品数 <= 100 首

**文件格式**: 同高产诗人

---

## 处理流程

### 步骤1: 数据加载与诗人汇总
1. 遍历所有 `poems_chunk_*.csv` 文件
2. 使用 `opencc` 将诗人名字从繁体转为简体
3. 按诗人汇总所有作品，记录 poem_id、poem_type、meter_pattern、words

### 步骤2: 生成 author_summary.json
1. 按作品数量降序排序
2. 保存为 JSON 文件

### 步骤3: 高产诗人处理（作品数 > 100）
1. 筛选高产诗人
2. 每人生成一个 chunk 文件
3. 统计格律模式分布
4. 统计用词频率（取前100）
5. 计算与其他诗人相似度

### 步骤4: 低产诗人处理（作品数 <= 100）
1. 筛选低产诗人
2. 多个诗人合并到一个 chunk 文件
3. 同样进行格律和用词统计

## 诗人相似度设计方案

### 方案: 基于用词的余弦相似度

#### 核心思路
将每位诗人的所有作品视为一个"文档"，计算其词频向量，然后使用余弦相似度衡量诗人之间的相似程度。

#### 实现步骤

1. **词频向量构建**
   - 对每位诗人的所有作品，合并所有分词结果
   - 统计每个词的出现次数
   - 构建词频向量（词 -> 频次）

2. **TF-IDF 加权**（可选优化）
   - 计算全局 IDF 值（每个词在多少诗人作品中出现）
   - 将原始词频转换为 TF-IDF 值
   - 公式: `TF-IDF = TF * IDF`
   - 作用: 降低常见词的影响，突出个性化用词

3. **余弦相似度计算**
   - 公式: `cosine(A, B) = (A · B) / (||A|| × ||B||)`
   - A 和 B 分别为两位诗人的词频/TF-IDF 向量
   - 值范围: [0, 1]，越接近1表示越相似

4. **相似诗人筛选**
   - 设定阈值: similarity > 0.5
   - 存储相似诗人列表

#### 优化考虑

- **词过滤**: 去除单字词、停用词（如"的"、"了"）
- **n-gram**: 可考虑加入2-gram、3-gram捕捉短语
- **向量化**: 可使用预训练中文词向量（如 BERT、Word2Vec）
