# 2026-03-15 诗人数据处理方案

## 输出设计

### author_summary.json
诗人按作品数量排序，包含作者名和诗词ID列表。

### author_chunk_*.json
**高产诗人**（>100首）：每人一个文件  
**低产诗人**（≤100首）：多人合并文件

包含：
- poem_type_counts - 诗体分布
- meter_patterns - 格律模式统计
- word_frequency - 用词频率（Top 100）
- similar_authors - 相似诗人列表

## 相似度计算方案

**基于TF-IDF的余弦相似度**

1. 合并诗人所有作品为"文档"
2. 构建词频向量
3. TF-IDF加权（可选）
4. 余弦相似度计算
5. 阈值筛选（>0.5）

## 关键优化

- 词过滤：去除单字词、停用词
- 繁简转换：统一使用简体
- 分块存储：高效加载
