### 快速示例

```python
import json
from pathlib import Path

# 加载唐诗
data_path = Path("data/chinese-poetry/全唐诗")
tang_poems = []
for json_file in data_path.glob("poet.tang.*.json"):
    with open(json_file, "r", encoding="utf-8") as f:
        tang_poems.extend(json.load(f))

print(f"共加载 {len(tang_poems)} 首唐诗")

# 统计作者
from collections import Counter
authors = Counter(p["author"] for p in tang_poems)
print(f"作者数: {len(authors)}")
print(f"作品最多的5位: {authors.most_common(5)}")
```

### 常见问题

**Q: 数据量太大怎么办？**
A: 先用小样本测试，如 `唐诗三百首.json`

**Q: 繁简转换有歧义？**
A: OpenCC 的 `t2s` 模式可处理大部分情况，特殊词汇需人工校对

**Q: 情感分析准确率低？**
A: 古文情感词典稀缺，建议先用 chinese-gushiwen 的赏析文本训练

**`src` 目录里大部分确实是垃圾代码**：

1. **未使用的技术**: HanLP、JioNLP、Plotly、Dash、Pyecharts 完全没有在代码中使用
