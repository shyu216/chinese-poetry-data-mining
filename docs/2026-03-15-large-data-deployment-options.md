# 2026-03-15 大数据部署方案对比

## 背景

`word_similarity_v2/` 约 1.8GB，超出 GitHub Pages 1GB 限制。

## 方案对比

| 方案 | 复杂度 | 成本 | 部署体积 | 访问速度 |
|------|--------|------|----------|----------|
| Git LFS + Release | 低 | 免费 | ~500MB | 中等 |
| 外部CDN (R2/S3) | 中 | 免费额度 | 1.8GB | 快 |
| 数据压缩+按需加载 | 高 | 免费 | ~400MB | 中等 |
| 功能降级/采样 | 低 | 免费 | ~100MB | 快 |

## 推荐方案

**Cloudflare R2**（方案B）
- 存储成本：~$0.03/月
- 请求：前1000万次/月免费
- 全球CDN加速
